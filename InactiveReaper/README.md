# InactiveReaper 

InactiveReaper 是一个基于 Hyper 框架的 QQ 机器人插件，专门用于清理1级的成员，并支持白名单保护机制。

## 功能特性

- 🔍 **一键扫描** - 自动列出群内等级 ≤ 1 的不活跃成员
- 🗑️ **批量清理** - 确认后一键清理成员
- 🛡️ **白名单保护** - 守护重要成员不被收割
- ⏱️ **确认超时机制** - 5分钟内未确认自动取消操作
- 📋 **权限管理** - 仅管理员可使用收割权限

## 命令列表

| 命令                    | 说明                          |
| ----------------------- | ----------------------------- |
| `#踢1级`                | 列出群内等级 ≤ 1 的不活跃成员 |
| `#yes`                  | 确认执行收割操作              |
| `#踢人白名单 添加 QQ号` | 将指定 QQ 加入白名单          |
| `#踢人白名单 移除 QQ号` | 从白名单中移除指定 QQ         |
| `#踢人白名单 列表`      | 查看当前白名单列表            |

### 1. 克隆或复制插件文件

# QQ名片点赞插件

这两个插件为QQ机器人提供了点赞功能，用户可以通过发送特定指令让机器人给自己的QQ名片点赞。

## 插件功能

### LikePlugin (赞我插件)
- 触发词: `赞我`
- 功能: 每次给用户QQ名片点赞10次，每日上限10次
- 查询功能: 发送`[前缀]点赞信息`查询今日点赞情况

### LikePlugin2 (超我插件)
- 触发词: `超我` 
- 功能: 每次给用户QQ名片点赞10次，每日上限10次
- 查询功能: 发送`[前缀]超信息`查询今日点赞情况

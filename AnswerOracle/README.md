# 📖 AnswerOracle — 答案之书插件

AnswerOracle 是一个趣味问答插件，用户向“答案之书”提问，插件会调用 API 返回一个神秘而随机的答案。适合群聊娱乐、占卜互动等场景。

---

## ✨ 功能特性

- ❓ 支持任意问题，自动提取用户提问内容
- 🔮 返回拟人化、带有神秘感的答案
- ⏰ 超时与异常处理，保证插件稳定
- 🔁 自动回复引用原消息，清晰友好

---

## 📦 安装与依赖

### 依赖安装

```bash
pip install aiohttp
```

### 命令格式

```
{reminder}答案之书 <你的问题>
```



### 示例

```
/答案之书 我今天会好运吗？
```



### 回复示例

```
📖 答案之书 🔮

❓ 我今天会好运吗？

✨ 答案：毫无疑问，是的。
```

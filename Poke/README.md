# Poke

一个可爱又强大的群聊戳一戳插件，支持指定用户和次数，带冷却保护机制。

---

## ✨ 功能特点

- 🎯 **@指定用户** 进行戳一戳
- 🔢 **自定义戳的次数**（最多 20 次）
- 🛡️ **冷却保护**（普通用户 10 分钟冷却）
- 👑 **管理员特权**（管理员无冷却限制）
- 🧠 **智能解析** – 自动识别被戳用户和次数
- 💬 **友好的提示信息** 和使用帮助

---

## 📦 依赖环境

- Python 3.8+
- Hyper Bot 框架
- 机器人后端支持群聊戳一戳 API

---

## 🚀 安装方法

1. 将 `Poke.py` 放入机器人的插件目录

2. 确保安装所需依赖：

pip install -r requirements.txt

# AutoSignMate 

> 一个智能的QQ群自动打卡插件，支持定时打卡、失败重试、状态查询等功能。

## ✨ 特性

- ⏰ **自动打卡**：每天00:00自动执行全群打卡
- 🔄 **智能重试**：失败自动重试，最多30次（至00:30）
- 📊 **状态追踪**：实时查看今日打卡进度和结果
- 🎮 **手动触发**：管理员可随时手动执行打卡
- 🔐 **权限控制**：支持多级管理员权限
- 📝 **失败记录**：记录失败群组及原因，便于排查

## 📦 安装

1. 将插件放入 `plugins` 目录

2. 安装依赖（如需要）：
```bash
pip install -r requirements.txt
```

## ⚠️ 注意事项

1. 确保Bot有群打卡权限（部分QQ协议可能需要特殊支持）
2. 打卡接口 `set_group_sign` 可能因QQ协议版本而异
3. 建议在00:00-00:30期间保持机器人在线
4. 重试间隔为60秒，避免频繁请求

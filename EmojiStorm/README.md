# EmojiStorm 🌩️

> 随机表情表态插件 — 对回复的消息瞬间点上20个不同的随机表情

## 功能描述

当用户使用 `表情` 命令（可自定义触发前缀）回复某条消息时，`EmojiStorm` 会自动为该目标消息随机添加 **20个不同的QQ表情**（从预设的100+个表情ID池中随机抽取），并且会**记录已点过的表情**，重复对同一条消息使用时不会重复添加相同的表情，直到该消息集满20个表情为止。

## 触发方式

`/表情`

> 注意：必须回复（Reply）一条消息才能使用此命令。  
> 触发前缀默认为 `/`，可根据框架配置中的 `reminder` 项修改。

## 使用示例

1. 用户A发送一条消息：“今天天气真好！”
2. 用户B回复该消息并发送：`/表情`
3. 机器人自动为目标消息点上20个随机表情（通过QQ表情表态接口）
4. 机器人回复：`好了🤓`

若重复对同一消息使用：
- 未满20个：继续补充未点过的随机表情
- 已满20个：提示“这条消息已经点满20个表情了”

## 表情池

插件内置了完整的QQ表情ID列表，包含：
- 基础表情（如微笑、笑哭、呲牙等）
- 特殊表情（如爱心、炸弹、咖啡等）
- Unicode扩展表情（如🍕、🍷、🎉等）

共计 **140+** 个表情ID。

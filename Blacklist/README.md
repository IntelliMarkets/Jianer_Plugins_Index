# Blacklist

一个黑名单插件

## ✨ 功能特性

- ✅ 拦截黑名单用户的所有消息
- ✅ 添加用户到黑名单（支持 @ 或输入 QQ 号）
- ✅ 从黑名单中移除用户
- ✅ 查看当前黑名单列表
- ✅ 仅 ROOT 用户可管理黑名单
- ✅ 支持群聊和私聊环境

## 🧩 命令说明

###所有命令需以配置文件中设置的 reminder 开头。
###添加用户到黑名单：
'添加用户黑名单 123456789'
'添加用户黑名单 @用户名'

###从黑名单移除用户：
'移除用户黑名单 123456789'
'移除用户黑名单 @用户名'

###查看当前黑名单：
'查看黑名单'
